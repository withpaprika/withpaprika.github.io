目标回顾

目标实现情况

  1. 学会游泳 - 蛙泳学会了，自由泳学了一半
  2. 跑马拉松 - 没中签
  3. 开展副业 - 没开始
  4. 阅读 - 上半年读书量太少了，不超过10本
  5. 多和朋友联系 - 很好

目标调整

  1. 继续练习蛙泳，同时尝试自学自由泳
  2. 下半年接着报马拉松
  3. 开展副业 - 整理可以输出的东西，逐渐开始运营小红书
  4. 阅读 - 多把时间花在读书上
  5. 多和朋友联系 - 继续保持

成就和亮点

主要成就

  1. 不再纠结于意义，生活和工作都变得清晰
  2. 工作上从执行层逐渐Level Up
  3. 不再害怕独处
  4. 心里对很多事情都看得更透彻了，比如Dating/社交/别人的看法
  5. 对于社交更游刃有余

高光事件

  1. 带父母一起去云南旅行
  2. 去日本了
  3. 去了很多城市，见了很多朋友：绍兴/昆明/香格里拉/南京/青岛/杭州/苏州
  4. 看了很多喜欢的演出：安溥/天爆/pglost/疫苗
  5. 获得了很满意的上影节经历

学习和成长

技能提升

  1. Obsidian使用
  2. 对日历和待办的定位和使用更清晰
  3. 对手帐本规划更清晰
  4. 学会蛙泳，不怕水了
  5. 音乐品味Level Up！听了很多优秀的音乐，知道自己真正喜欢的是什么样的了
  6. 数据开发技能Level Up！学会实时数仓、数据仓库

经验教训

  1. 面对冲突时一定不要陷入自证，让时间和事实说话
  2. 我创造我喜欢的生活，创造新鲜的体验和经历，努力去获得我喜欢的事情的能力和自由，这就是我生命最大的意义

挑战和问题

面临的挑战

  1. 陷入虚无主义
  2. 对有毒关系的处理
  3. 懒得运动
  4. 作息不好

解决方案和应对措施

  1. 把自己关起来拼命思考，决心活在当下
  2. 保持界限
  3. 从有趣的事情做起，比如游泳/滑板/攀岩/徒步
  4. 无论做什么，都要保证自己充足的休息时间

时间管理和效率

时间使用

完全没有在意自己时间管理

效率提升

下半年要开始做任务、时间、精力管理，能做更多事情吧

生活质量和Work Life Balance

生活质量

在吃饭方面不太讲究，在其他方面都很好 - 要在乎自己的健康生活呀

Work Life Balance

一开始是更重视Life，想要尽可能逃避Work，现在已经很释然，Work也是我Life的一部份，尽量让这一部分也变得有趣，我便不会再排斥

人际关系

关系维护

  1. 生活：过滤掉了一些有毒的关系，留下了我喜欢的朋友，认识了新的朋友，和老朋友也一直在保持联系和见面
  2. 工作：和同事保持良好关系，比去年好；和领导关系，比去年好

支持网络

  1. 家人：都好好的就是最大的支持
  2. 小猫：无条件的爱
  3. 从小的老朋友：尽量去理解，无法共情的时候就不要对这些人倾诉
  4. 上海的朋友：主打迅速可以联系到，能获得第一时间的支持
  5. 领导：在职场上的问题，领导兜底和支持

财务状况

财务回顾

除了前两个月云南旅行花的太多，其余时间每个月都有不少结余，哪怕去了一趟日本，当月结余也是正的。
每周定投100元美股基金，形势不错，收益率在9%左右。另外买了短期债基和其他的QDII基金，目前理财方面的储蓄有1.3w左右。下半年因为要长期旅行，还有Fuji Rock，不确定能攒到多少，但争取到年底有4w结余吧。

财务计划

下半年两项重要支出：Fuji Rock；9月WFH；总预算3w
日常开销6w
半年期望攒2.7w；年底期望结余4w；

健康与幸福

身体健康

  1. 饮食 - 完全没注意，三、四月自己做了一段时间的饭，开心、健康又省钱，但是没有坚持下来
  2. 锻炼 - 不定期游泳，经常走路，但完全没有达到目标。
  3. 医疗 - 没去过医院，还挺健康的

心理健康

4月开始重新每天一片帕罗西汀，没有去医院，纯粹是那时候自己状态不好开始吃药。随后的日子还不错，但完全不想去复诊，就这么吃下去也可以吧。

短期计划

职业发展

  • 技术：系统学习大数据，提升数据开发及相关技术能力
  • 项目管理：主动参与新项目，用成绩证明自己的能力

个人成长

  • 新技能：1）学习英语口语 2）学习Charleston
  • 自我意识：吾日三省吾身，并记录

兴趣爱好

  • 新鲜体验：1）Fuji Rock；2）数字游民一个月体验
  • 艺术和音乐：

  • 健康管理：养成锻炼习惯，不限运动种类，每周3次
  • 护肤变美：认真护肤！！！建立护肤SOP

人际关系

  • 社交拓展：认识新朋友，随缘！
  • 维护关系：和现有的朋友保持联系、提供支持

  • 学习投资：

  • 运营小红书：目标年底累计1k粉

长期战略（VISION）

职业发展

  • 愿景： 高度自主性和创新性，同时有保障
  • 成为Data领域的专家或领导者，探索更高层次的职位
  • 扩展跨领域技能，like项目管理、团队领导、战略规划

个人成长

  • 愿景： 增强主体性、自我意识、技能，提升生活质量
  • 全面发展，学习社会学、心理学，深化对自我和世界的理解
  • 健康生活，维持身体健康和心理健康，这是一切的基石

兴趣爱好

  • 愿景： 深入发展个人兴趣，丰富生活体验
  • 持续参与音乐节和艺术活动，扩大这方面的圈子
  • 制定更多更丰富的旅行计划，用脚步丈量世界

  • Network
  • 对于朋友和家人，组织聚会和旅行，提供支持和帮助，保持紧密联系
